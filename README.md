## Key Features

- User can Login with google account
- User can send and recieve messages in different channels
- User can see members of the channels

## How To Use

```bash
# Clone this repository
$ git clone https://github.com/TashfeenRao/chat-app

# Go into the repository
$ cd chat-app

# Install dependencies
$ npm install

# Run the app
$ npm start
```
